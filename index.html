1.1 — Sets of Numbers & Inequalities
Number Sets Chain
ℕ ⊂ W ⊂ ℤ ⊂ ℚ ⊂ ℝ
  • ℕ = {1, 2, 3, …} — Natural (counting, no zero)
  • ℤ = {…, −2, −1, 0, 1, 2, …} — Integers
  • ℚ = {p/q | q≠0} — Rational (terminating/repeating decimals)
  • ℝ — All reals, including irrational numbers (π, √2, e)
Intervals
(a, b) : a < x < b
[a, b] : a ≤ x ≤ b
Sign Rule — multiply/divide by negative → flip the inequality
Absolute Value
|a| = a if a ≥ 0,  −a if a < 0
Triangle Inequality:  |a+b| ≤ |a| + |b|
|x − a| < r  ⟹  a − r < x < a + r
1.2 — Functions
Definition & Tests
f : Domain → Range
  • Domain — all valid inputs (x-values)
  • Range — all possible outputs (y-values)
  • VLT — every vertical line crosses the graph at most once
Types of Functions
Linear:  f(x) = mx + b
Power:  f(x) = xⁿ
Root:  f(x) = ⁿ√x  (√x domain: [0, ∞))
Rational:  f(x) = p(x)/q(x),  q(x) ≠ 0
Operations & Composition
(f ± g)(x) = f(x) ± g(x),  D = Df ∩ Dg
(f / g)(x) = f(x) / g(x),  g(x) ≠ 0
(f ∘ g)(x) = f(g(x))
Order matters: f∘g ≠ g∘f in general

1.3 — Inverse Functions
One-to-One Condition
f(x₁) = f(x₂)  ⟹  x₁ = x₂
Horizontal Line Test (HLT): every horizontal line intersects the graph at most once → one-to-one
Finding f⁻¹(x) — 3 Steps
  • Write y = f(x)
  • Swap x and y
  • Solve for y — that is f⁻¹(x)
Domain(f⁻¹) = Range(f)
Range(f⁻¹) = Domain(f)
Graph of f⁻¹ is the reflection of f across y = x
Cancellation Laws
f⁻¹(f(x)) = x  for x in domain of f
f(f⁻¹(x)) = x  for x in domain of f⁻¹
1.4 — Trig Functions & Inverses
Angles & ASTC Rule
180° = π rad
s = rθ
ASTC — which trig functions are positive per quadrant:
Q1: All  ·  Q2: Sin  ·  Q3: Tan  ·  Q4: Cos
Pythagorean Identities
sin²θ + cos²θ = 1
1 + tan²θ = sec²θ
cot²θ + 1 = csc²θ
Inverse Trig — Domains & Ranges
sin⁻¹(x) : [−1, 1] → [−π/2, π/2]
cos⁻¹(x) : [−1, 1] → [0, π]  (decreasing)
tan⁻¹(x) : ℝ → (−π/2, π/2)

2.1 — Definition of the Limit
Core Concept
lim x→a f(x) = L
f(x) gets arbitrarily close to L as x approaches a — without x actually equaling a
Formal ε-δ Definition
∀ε > 0  ∃δ > 0 :  0 < |x−a| < δ  ⟹  |f(x)−L| < ε
One-Sided Limits & Existence
lim x→a⁺ — approach from the right (x > a)
lim x→a⁻ — approach from the left (x < a)
Limit exists  ⟺  lim a⁺ = lim a⁻ = L
2.2 — Limit Laws
Basic Laws
lim(f ± g) = lim f ± lim g
lim(f · g) = lim f · lim g
lim(f / g) = lim f / lim g  (lim g ≠ 0)
Handling 0/0 Indeterminate Forms
Factoring — cancel the common zero factor
Rationalize — multiply by conjugate
LCD — combine fractions, then simplify
Squeeze Theorem
g(x) ≤ f(x) ≤ h(x)  and  lim g = lim h = L  ⟹  lim f = L
lim x→0  sin(x) / x  = 1

2.3 — Limits Involving Infinity
Limits at ±∞
lim x→+∞ 1/x = 0
lim x→−∞ eˣ = 0
Rational functions — highest-degree term dominates:
deg top > bot → ±∞  |  deg top < bot → 0  |  equal → ratio of leading coefficients
Asymptotes
Horizontal:  lim x→±∞ f(x) = L  ⟹  y = L
Vertical:  lim x→a f(x) = ±∞  ⟹  x = a
Vertical asymptotes occur at zeros of denominator — simplify first
2.4 — Continuity of Functions
3 Conditions for Continuity at x = a
①  f(a) is defined
②  lim x→a f(x) exists
③  lim x→a f(x) = f(a)
All three conditions must hold simultaneously
Types of Discontinuity
  • Removable — limit exists but ≠ f(a), or f(a) undefined
  • Jump — lim a⁺ ≠ lim a⁻ (common in piecewise functions)
  • Infinite — limit equals ±∞ (at vertical asymptotes)
Intermediate Value Theorem (IVT)
f continuous on [a,b],  f(a) ≤ k ≤ f(b)  ⟹  ∃c ∈ (a,b) : f(c) = k
If f changes sign on [a,b] → guaranteed root exists in (a,b)
